# SWStarter

## Overview
A React application using SWAPI (Star Wars API) to search for character and movie information. 

## Get Started

### clone the repo
```
git clone https://github.com/brittani1128/swstarter.git
```

### install dependencies
```
cd swstarter
yarn install
yarn start
```

### install React dependencies
```
cd client
yarn install
yarn start
```
